# 🛡️ 1-ci Gün: Etik Hakerliyin Əsasları (Foundation)

## 🧐 Etik Hakerlik Nədir?
Etik hakerlik, zərərli hakerlərin istifadə edə biləcəyi hücum növlərini simulyasiya etməkdir. Onu qanunsuz hakerlikdən ayıran ən vacib fərq: **hədəfin icazəsi ilə** və sistemin təhlükəsizliyini artırmaq məqsədilə həyata keçirilməsidir.

---

### 🎯 Əsas Məqsəd
Etik hakerin əsas hədəfi zərərli aktyorlar tərəfindən istismar edilə bilən zəiflikləri müəyyən etmək və onları aradan qaldırmaqdır. Məqsəd sistemləri daha **möhkəm** və **davamlı** etməkdir.

### ⚙️ Metodologiya
Etik hakerlər real hücum ssenarilərinə əsaslanaraq testlər aparırlar:
* **Ağ papaq (White Hat):** Qanuni və etik çərçivədə fəaliyyət.
* **Zəiflik Qiymətləndirilməsi:** Sistemdəki boşluqların tapılması.
* **Metodiki Test:** Formal və sistemli yoxlama üsulları.

### 🛠️ Tələb Olunan Bacarıqlar
Bu sahədə peşəkarlaşmaq üçün aşağıdakı biliklər vacibdir:
1. **Proqramlaşdırma:** Xüsusilə Linux və Unix-in əsası olan **C dili** kimi dilləri anlamaq.
2. **Əməliyyat Sistemləri:** Windows və Linux-un dərinlikləri.
3. **Şəbəkə (Networking):** Məlumatın necə daşındığını bilmək.
4. **Təhlükəsizlik Alətləri:** Nmap, Metasploit və digər peşəkar tool-lar.

---

> 💡 **Mühüm Qeyd:** Etik hakerlik böyük məsuliyyət tələb edir. Bu biliklər yalnız sistemləri qorumaq və təhlükəsizliyi təmin etmək üçün istifadə olunmalıdır.
