# 🌐 8-cü Gün: Əməliyyat Sistemləri


🌟 Giriş: Görünməz Qəhrəmanla Tanışlıq
Sən hər gün telefonunu və ya noutbukunu açırsan və hər şey "sadəcə işləyir": proqramlar açılır, fayllar əlçatan olur, musiqi çalınır. Bəs bütün bunları mümkün edən nədir?

Fiziki komponentləri (hardware) artıq tanıyırıq. İndi isə bütün bunları bir-birinə bağlayan görünməz təbəqəni — Əməliyyat Sistemini (ƏS) kəşf etməyin vaxtıdır.

#  🎭 Ssenari: Miras Qalan Kompüter
Dostun sənə köhnə kompüterini hədiyyə edib. O, cihaz haqqında çox şey xatırlamır, sadəcə "yaxşı işləyirdi" deyir. Onu yeniləmək, satmaq və ya yeni kiber təhlükəsizlik layihən üçün istifadə etmək qərarına gəlməzdən əvvəl, əlində tam olaraq nəyin olduğunu anlamalısan.

# 🏛️ 1. Kompüter Sisteminin Daxili Quruluşu: Kompüter Növləri
Əməliyyat sisteminə keçməzdən əvvəl, onun hansı "mühitlərdə" yaşadığını bilməliyik. Kompüterlər ölçülərinə və güclərinə görə fərqlənirlər:

Fərdi Kompüterlər (PC): Gündəlik istifadə etdiyimiz noutbuklar və masaüstü cihazlar.

Serverlər: İnternet saytlarını 24/7 işlək saxlayan güclü cihazlar. Adətən ekranları olmur, uzaqdan idarə edilirlər.

Mobil Cihazlar: Smartfonlar və planşetlər (kiçik ölçü, batareya qənaəti önəmlidir).

Quraşdırılmış Sistemlər (Embedded Systems): Avtomobillər, smart TV-lər və ya paltaryuyan maşınlar kimi xüsusi funksiyalı cihazlar.

Virtual/Bulud Sistemləri: Fiziki bir yerdə olmayan, internet üzərindən istifadə etdiyimiz "bulud" kompüterləri.

# 🎮 2. ƏS ilə Ünsiyyət: GUI və CLI
Kompüterlə necə danışırıq? İki əsas yolumuz var:

A) Qrafik İstifadəçi İnterfeysi (GUI)
Bu, hamımızın öyrəşdiyi yoldur: qovluq ikonları, pəncərələr, menyular.

Bənzətmə: Naviqasiya tətbiqindən istifadə etmək kimidir. Getmək istədiyin yerin şəklini seçirsən və proqram səni aparır. Yazmağa ehtiyac yoxdur, sadəcə klikləyirsən.

B) Əmr Sətri İnterfeysi (CLI)
Bura xüsusi mətn əmrləri yazaraq kompüteri idarə etdiyimiz yerdir (məsələn, Terminal və ya CMD).

Bənzətmə: GPS koordinatlarını əllə daxil etmək kimidir. Daha çətindir, çünki koordinatı bilməlisən, amma birbaşa və daha sürətlidir. Peşəkar siber təhlükəsizlik mütəxəssisləri və proqramçılar CLI-ya üstünlük verirlər.

# 🗺️ 3. Əməliyyat Sistemləri Dünyası (Real World OS)
Niyə bu qədər çox ƏS var? Çünki hər birinin öz işi var: bir server stabillik istəyir, bir telefon isə batareyaya qənaət.

💻 Masaüstü (Desktop)
Windows: Ən çox yayılmış (Windows 10, 11).

macOS: Apple cihazları üçün şık və stabil.

Linux: Açıq mənbəli, fərdiləşdirilə bilən (Ubuntu, Fedora).

🖥️ Server
Linux (Ubuntu Server, Red Hat): İnternetin böyük hissəsi burada işləyir.

Windows Server: Korporativ şəbəkələr üçün.

📱 Mobil
Android: Dünyada ən çox istifadə edilən mobil ƏS.

iOS: iPhone və iPad-lər üçün qapalı ekosistem.

🛰️ Xüsusi Cihazlar (Embedded & IoT)
Real-Time OS (RTOS): Təyyarə idarəetməsi kimi saniyənin mində biri qədər sürətli cavab tələb edən sistemlər.

# 🔍 4. Araşdırmaya Davam: İstintaq Başlayır!
Dostunun verdiyi kompüteri yandırırsan və görürsən ki, o, Linux Ubuntu ilə işləyir. Gəl, bu cihaz haqqında daha çox məlumat toplayaq.

🚀 Praktik Tapşırıq (Challenge)
Sən artıq bilirsən ki, bu cihaz Ubuntu-dur. İndi isə həm GUI, həm də CLI vasitəsilə sistem məlumatlarını öyrən:

GUI yolu: "Settings" -> "About" bölməsinə daxil ol. Orada cihazın adı, RAM miqdarı və prosessor modelini qeyd et.

CLI yolu: Terminalı aç (Ctrl + Alt + T) və bu əmri yaz:
uname -a



# 🧠 Niyə Bu Qədər Çox ƏS Var?
Çünki heç bir ƏS mükəmməl deyil. Bir noutbuk istifadəçi dostu olmalıdır, amma bir server 1 il boyunca heç sönmədən işləməlidir. Mobil cihazlar batareyanı qorumalıdır, quraşdırılmış sistemlər isə çox kiçik yaddaşla işləməlidir.

Növbəti addımda: Linux Terminalının (CLI) dərinliklərinə enəcəyik və faylları kodlarla necə idarə edəcəyimizi öyrənəcəyik!
